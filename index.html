Etiqueta Descripción
Head Especifica la cabecera del resto del archivo.
Title el navegador identifica el titulo de cada pagina y lo muestra en la parte superior, en el borde de la pantalla junto con el icono del programa.
Body Encierra el resto del documento, el contenido.
p Directiva que se utiliza para definir un párrafo.
br indica que el siguiente texto va a empezar a escribirse en la siguiente línea, sin dejar líneas en blanco entre texto y texto..
Center Esta etiqueta centrara todo elemento que se encuentre en su apertura y cierre.
b convierte el texto en negrita.
Marquee sirve para conseguir texto en movimiento, el cual aparecerá desplazándose, por defecto, de derecha a izquierda.
Font Define que características tendrá el texto que aparezca en la página, ya sea la fuente tipográfica, el color o el tamaño.
Footer Coloca el texto como un pie de página.

Etiqueta Descripción Sintaxis
Colors A través del Lenguaje de estilo en cascada CSS se puede cambiar el color de los elementos HTML.
Backgrounds A través del Lenguaje de estilo en cascada CSS se puede configurar el color de fondo de algunos elementos HTML.
borders A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de los bordes para algunos elementos HTML.
margins A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de las márgenes para algunos elementos HTML.
fonts A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de fuente para algunos elementos HTML.
Height,width A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de ancho y alto para algunos elementos HTML.
links A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de estilo de un link.
lists A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de estilo de las listas HTML.
tables A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de estilo de las tablas en HTML.
position A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de posicionamiento de un elemento HTML. (Static, relative, fixed, absolute, sticky)
Overflow A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de desbordamiento de un elemento HTML cuando su contenido es muy grande. (Visible, hidden, scroll, auto)
float A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de posicionamiento flotante de un elemento HTML. (left, right, none, inherit)
