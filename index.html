Etiqueta Descripción
Head Especifica la cabecera del resto del archivo.
Title el navegador identifica el titulo de cada pagina y lo muestra en la parte superior, en el borde de la pantalla junto con el icono del programa.
Body Encierra el resto del documento, el contenido.
p Directiva que se utiliza para definir un párrafo.
br indica que el siguiente texto va a empezar a escribirse en la siguiente línea, sin dejar líneas en blanco entre texto y texto..
Center Esta etiqueta centrara todo elemento que se encuentre en su apertura y cierre.
b convierte el texto en negrita.
Marquee sirve para conseguir texto en movimiento, el cual aparecerá desplazándose, por defecto, de derecha a izquierda.
Font Define que características tendrá el texto que aparezca en la página, ya sea la fuente tipográfica, el color o el tamaño.
Footer Coloca el texto como un pie de página.

Etiqueta Descripción Sintaxis
Colors A través del Lenguaje de estilo en cascada CSS se puede cambiar el color de los elementos HTML. h1 {color: rgb(38, 99, 229 ); }
Backgrounds A través del Lenguaje de estilo en cascada CSS se puede configurar el color de fondo de algunos elementos HTML.h1 { background-color: rgb(38, 99, 229); }
borders A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de los bordes para algunos elementos HTML.h1 {border: groove; border-color: rgb(0, 0, 129); }
margins A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de las márgenes para algunos elementos HTML.div {margin: 80px; border: 1px solid #4CAF50; }
fonts A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de fuente para algunos elementos HTML.h1 {font-family: sans-serif; }
Height,width A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de ancho y alto para algunos elementos HTML.div {width: 500px; height: 100px; background-color: rgb(0, 0, 129); }
links A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de estilo de un link./* Link no visitado */a:link { color: red; }
lists A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de estilo de las listas HTML.ul.a {list-style-type: square; }
tables A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de estilo de las tablas en HTML.table, td, th {border: 1px solid #ddd; text-align: left; }
position A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de posicionamiento de un elemento HTML. (Static, relative, fixed, absolute, sticky)div.relative {position: relative; width: 500px; height: 230px; border: 3px solid #83A4EB; }
Overflow A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de desbordamiento de un elemento HTML cuando su contenido es muy grande. (Visible, hidden, scroll, auto)div {background-color: #eee; width: 210px; height: 55px; border: 1px dotted black; overflow: auto; }
float A través del Lenguaje de estilo en cascada CSS se pueden configurar las propiedades de posicionamiento flotante de un elemento HTML. (left, right, none, inherit)img {float: right; }
